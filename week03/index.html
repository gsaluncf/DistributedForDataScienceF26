Week 3 Overview (Tue Feb 10, Thu Feb 12)

Build and deploy a minimal Python API on AWS using API Gateway and Lambda, with persistence in DynamoDB. Learn local testing, logging, and basic observability.

Tuesday — Foundations

  • Lambda execution model; event structure; cold starts.
  • API Gateway (HTTP APIs) basics and integrations.
  • CloudWatch logs and metrics for Lambda.

Thursday — Persistence with DynamoDB

  • Table design: partition/sort keys, access patterns.
  • Throughput (on-demand vs provisioned), hot partitions.
  • Integrate Lambda handlers with DynamoDB (put/get).

Readings

  • Create your first Lambda
  • API Gateway — HTTP APIs
  • CloudWatch logs for Lambda
  • DynamoDB core components

🧹 Purge Messages

Instantly delete all messages from both queues without deleting the queues themselves. Use when you want a clean slate between rounds.

🧪 Test Round-Trip

Send a test order to pizza-orders, then immediately receive it, "cook" it, post to pizza-results, and verify the full pipeline works end-to-end.

🔧 Drain Stuck Messages

Receive and delete all visible messages from a queue. Use when messages are stuck and purge is on cooldown (AWS enforces 60s between purge calls).

💣 Delete Queues

Permanently delete both queues. AWS enforces a 60-second cooldown before you can recreate queues with the same name. Use only for end-of-class cleanup.

🔄 Full Reset (Nuclear Option)

Delete both queues, wait 65 seconds, then recreate them fresh. Use when everything is broken and you need a clean restart. This takes ~70 seconds.

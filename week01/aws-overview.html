What is AWS?

Amazon Web Services (AWS) is the world's leading cloud computing platform. In this course, we'll use AWS to build and deploy real distributed systems.

Why AWS for This Course?

AWS provides the perfect environment for learning distributed systems because:

  • Real-world platform: Used by millions of companies including Netflix, Airbnb, and NASA
  • Distributed by design: Services like S3, Lambda, and DynamoDB embody the concepts we study (scalability, fault tolerance, consistency)
  • Free for students: AWS Educate provides $100 in credits with no credit card required
  • Industry skills: AWS experience is highly valued in data science and engineering roles

Key Services We'll Use

Service What It Does
S3 Object storage - stores files with automatic replication across data centers
Lambda Serverless compute - run code without managing servers
DynamoDB NoSQL database - distributed key-value store with configurable consistency
SQS Message queue - enables asynchronous communication between services
